LEVEL LANGUAGE

 ภาษาใดๆที่ผู้ใช้งานใช้สื่อสารกับคอมพิวเตอร์ หรือคอมพิวเตอร์ด้วยกัน แล้วคอมพิวเตอร์สามารถ ทำงานตามคำสั่งนั้นได้ คำนี้มักใช้เรียกแทนภาษาโปรแกรม แต่ความเป็นจริงภาษาโปรแกรมคือส่วนหนึ่งของภาษาคอมพิวเตอร์เท่านั้น และมีภาษาอื่นๆที่เป็นภาษาคอมพิวเตอร์เช่นกัน

  LOW LEVEL LANGUAGE
    (ภาษาระดับต่ำ)
- ภาษาเครื่อง (Machine Language) ที่ประกอบด้วยตัวเลขเฉพาะ 0 และ 1 เท่านั้น
- เครื่องคอมพิวเตอร์สามารถท าความเข้าใจได้ทันที
- การเขียนโปรแกรมค่อนข้างยุ่งยากและไม่สะดวก

  HIGH LEVEL LANGUAGE
    (ภาษาระดับสูง)
- พัฒนาให้มีรูปแบบใกล้เคียงกับภาษาของมนุษย์เรียกว่า ภาษาระดับสูง (High-Level Language)
- มีกลุ่มคำภาษาอังกฤษที่เข้าใจง่ายขึ้น
- เป็นภาษาเชิงกระบวนการหรือ Procedural Language ทำงานเป็นขั้นตอน เรียงตามลำดับคำสั่งที่เขียน
- เขียนโปรแกรมที่ซับซ้อนได้มากขึ้น แต่ก็ยังยุ่งยากอยู่บ้าง
- หากเป็นโปรแกรมขนาดใหญ่ต้องอาศัยประสบการณ์และความชำนาญ

PROGRAM DESIGN

PROGRAM DEVELOPMENT CYCLE

 ก่อนที่จะเริ่มทำงาน ควรมีการวางแผนก่อนการเขียนโปรแกรม มีวิธีที่ใช้ในการวางแผนก่อนการเขียนโปรแกรมอยู่หลายวิธี เช่น pseudocode และผังงานเพื่อสร้างรูปแบบของโปรแกรม

  DESIGNING A PROGRAM
 นักเขียนโปรแกรมควรที่จะออกแบบรากฐานของโปรแกรมให้ดีตั้งแต่ตอนแรกๆ เพื่อลดปัญหาที่จะตามมาในอนาคต
 กระบวนการออกแบบโปรแกรมสามารถสรุปได้ 2 ประการดังนี้
1. เข้าใจงานที่โปรแกรมจะทำ
2. กำหนดขั้นตอนที่ต้องดำเนินการเพื่อดำเนินงาน

FLOWCHART

INPUT PROCESS OUTPUT

 โปรแกรมคอมพิวเตอร์จะดำเนินการตามขั้นตอน 3 ขั้นตอนนี้
1. Input
2. Process
3. Output
 โดยการที่จะรับข้อมูลเข้ามา และนำไปดำเนินการด้วยวิธีการต่างๆ เช่น วิธีการทางคณิตศาสตร์ และจะนำผลลัพธ์ออกมาแสดงเป็นเอาต์พุต

VARIABLES

 ตัวแปร (Variable) คือชื่อหรือเครื่องหมายที่กำหนดขึ้นสำหรับใช้อ้างถึงค่าที่เก็บในหน่วยความจำ ตัวแปรจะมีชื่อ (Identifier) สำหรับใช้ในการอ้างถึงข้อมูลของมัน ในการเขียนโปรแกรม ค่าของตัวแปรสามารถที่จะกำหนดได้ใน run-time หรือเปลี่ยนแปลงอยู่ตลอดเวลาในขณะที่โปรแกรมทำงาน (Executing)
