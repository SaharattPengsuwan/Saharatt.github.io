Introduction

SEQUENCE TYPES

The fundamental sequence types : list และ tuples ทั้ง list และ tuples เป็นลำดับที่สามารถเก็บข้อมูลประเภทต่างๆ ได้ ความแตกต่างระหว่าง list และ tuples นั้นง่ายมาก
• List เปลี่ยนแปลงได้ ซึ่งหมายความว่าโปรแกรมสามารถเปลี่ยนเนื้อหาได้
• Tuple นั้นไม่สามารถเปลี่ยนแปลงได้ ซึ่งหมายความว่าเมื่อสร้างแล้ว เนื้อหาจะไม่สามารถเปลี่ยนแปลงได้

LISTS

List เป็นสิ่งที่สามารถเปลี่ยนแปลงข้อมูลด้านใน list ในระหว่างการดำเนินการของโปรแกรมได้ list เป็นโครงสร้างข้อมูลแบบไดนามิก นั้นคือข้อมูลใน list สามารถถูกเพิ่มและลบได้ ด้วยการทำดัชนี การแบ่งส่วน และวิธีต่างๆ เพื่อทำงานกับ list ในโปรแกรม
• List : คือประเภทของข้อมูลที่มีข้อมูลหลายรายการในตัว List
 - Format :
 list = [item1, item2, etc.]
 - List สามารถใส่ข้อมูลได้หลาย
 ประเภท
• ฟังก์ชัน print สามารถใช้แสดงรายการทั้งหมดได้
• ฟังก์ชั่น list() สามารถแปลงข้อมูลบางประเภทเป็น list

LIST METHODS

• append(item) : เพิ่มออบเจ็ค x เข้ามายัง List
• index(item) : ค้นหาตำแหน่งของออบเจ็ค x
• insert(index,item) : ใส่ออบเจ็ค x เข้ามายัง List ที่ตำแหน่ง i
• sort(): : เรียงข้อมูลภายใน List จากน้อยไปมาก
• remove(item): : นำออบเจ็ค x อันแรกที่พบใน List ออก
• reverse(): : ย้อนกลับลำดับของ List
• del : : ลบข้อมูลออกจาก list
• min and max : ส่งค่าคืนข้อมูลที่มีค่าต่ำสุดและสูงสุดใน list

Two-Dimensional Lists

TWO-DIMENSIONAL LISTS

• Two-dimensional list : list ที่มี list อื่นเป็นองค์ประกอบ
 - เป็นที่รู้จักกันในนาม nested list
 - เป็น list ที่มีแถวแนวตั้งและแถวแนวนอน
 - มีประโยชน์สำหรับการทำงานกับข้อมูลหลายชุด
• ในการประมวลผลข้อมูลใน list two-dimensional จำเป็นต้องใช้สองดัชนี
• โดยทั่วไปแล้วจะใช้การวนซ้ำซ้อนเพื่อประมวลผล

Introduction TUPLES

TUPLES

• Tuple : ลำดับที่ไม่เปลี่ยนรูป
 - คล้ายกับ list มาก
 - เมื่อสร้างขึ้นแล้วจะไม่สามารถเปลี่ยนแปลงได้
 - format :
  tuple_name = (item1, item2)
 - Tuples รองรับการดำเนินการตาม list
  - การทำดัชนีตัวห้อยสำหรับการดึงองค์ประกอบ
  - วิธีการต่างๆ เช่น index
  - ฟังก์ชันในตัว เช่น len, min, max
  - Slicing expressions
  - The in, +, and * operators
• สิ่งที่ Tuple ไม่สามารถทำได้
 - เพิ่มข้อมูล
 - ลบข้อมูล
 - ค้นหาข้อมูล
 - ย้อยกลับลำดับของข้อมูล
 - เรียงลำดับของข้อมูล
• ข้อดีของ Tuple มีดีกว่า List
 - การประมวลผล Tuple นั้นเร็วกว่าการประมวลผล list
 - Tuple ปลอยภัยกว่า
 - การดำเนินการบางอย่างใน Python ต้องใช้ tuples
• Tuple สามารถแปลงเป็น List ได้
• List สามารถแปลงเป็น Tuple ได้
