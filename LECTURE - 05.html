Introduction to Functions

ฟังก์ชันคือกลุ่มของคำสั่งที่มีอยู่ภายในโปรแกรมสำหรับปฏิบัติงานใดงานหนึ่งโดยเฉพาะ

 Functions
• ฟังก์ชันคือส่วนของโค้ดที่ใช้ซ้ำได้
• ฟังก์ชันมีชื่อและต้องประกาศ
ชื่อฟังก์ชันนั้นดำเนินการฟังก์ชั่นนั้นๆ
• ฟังก์ชันสามารถรับและส่งคืนข้อมูลได้








FUNCTIONS NAMES

 Function naming rules
• ไม่สามารถใช้คำสำคัญเป็นชื่อฟังก์ชันได้
• ไม่สามารถตั้งชื่อโดยใช้ช่องได้
• อักขระตัวแรกต้องเป็นตัวอักษรหรือขีดล่าง
• อักขระอื่นๆทั้งหมดต้องเป็นตัวอักษร ตัวเลข หรือขีดล่าง
• อักขระตัวพิมพ์ใหญ่และตัวพิมพ์เล็กต่างกัน ชื่อของฟังก์ชันเหมือนกันแต่ตัวอักษรพิมพ์เล็กพิมพ์ใหญ่ต่างกัน ถือเป็นคนละฟังก์ชันกัน



BENEFITS OF MODULARISING A PROGRAM WITH FUNCTIONS

 Function naming rules
• เข้าใจง่ายขึ้นเมื่อแบ่งออกเป็นฟังก์ชัน
• เขียนครั้งเดียวเพื่อดำเนินการและดำเนินการเมื่อใดก็ได้ที่จำเป็น
• สามารถทดสอบแต่ละฟังก์ชันในโปรแกรมทีละรายการ แก้ไขข้อผิดพลาดได้ง่ายขึ้น
• ไลบรารีของฟังก์ชันสามารถรวมเข้ากับแต่ละโปรแกรมที่ต้องการได้
• สามารถมอบหมายงานให้แต่ละคนเขียนโปรแกรมที่ต่างกันและนำมารวมกันได้
