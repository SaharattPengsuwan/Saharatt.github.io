BASIC STRING METHODS

strip : strip method เป็น method ที่เอาไว้ลบช่องว่างก่อนและหลังของ string
join() : การ join จะอนุญาตให้เรานำ string ที่อยู่ใน Tuple, List, Set มาเชื่อมเข้าด้วยกันได้
upper() : การทำให้ชุด String ทั้งหมดเป็นตัวพิมพ์ใหญ่
lower() : การทำให้ชุด String ทั้งหมดเป็นตัวพิมพ์เล็ก
split() : สำหรับ split เป็นการแยกนะคับตามตัวเลย ซึ่งถ้าเรากำหนดคำแยกใน argument มันก็จะแยกเมื่อเจอเครื่องหมายนั้น แต่ถ้าไม่กำหนดอะไรเลย มันจะแยกด้วย space

SEARCH AND REPLACE METHODS

endswith() : ใช้สำหรับตรวจสอบว่า สตริงนั้น ๆ ลงท้ายด้วยตัวอักษรหรือข้อความที่ระบุหรือไม่ โดยจะคืนค่าเป็น True หรือ False
find() : คำสั่ง หรือฟังก์ชันสำหรับค้นหาข้อความ หรือตัวอักษรที่ต้องการจากข้อมูล หรือประโยค
replace() : ใช้สำหรับแทนที่ คือ ถ้าเจอ string ตัวนี้ ก็ให้แทนที่ด้วยตัวนี้
startswith() : ใช้สำหรับตรวจสอบว่าสตริงนั้นขึ้นต้นด้วยคำตามที่ระบุหรือไม่ ถ้าใช่จะคืนค่าเป็น True ถ้าไม่ใช่จะคืนค่าเป็น False
