SET

Set Methods and Useful Built-in Functions

• Set : เป็นสิ่งที่ใช้สำหรับเก็บข้อมูลที่ไม่ซ้ำกันและดำเนินการทางคณิตศาสตร์เกี่ยวกับ Set ได้
 - ข้อมูลทั้งหมดต้องไม่ซ้ำกัน
 - ข้อมูลใน Set จะไม่เรียงลำดับ
 - ข้อมูลภายในสามารถเป็นข้อมูลที่ประเภทต่างกันได้
• function len : ใช้เพื่อนับจำนวนองค์ประกอบที่อยู่ใน Set
• method add : เพื่อเพิ่มข้อมูลใน Set
• method update : เป็นการนำข้อมูลของ set อื่นๆไปใส่ในอีก set หนึ่ง
• method remove : ลบข้อมูลออกจาก Set
• method discard : สำหรับลบข้อมูลออกจาก Set ถ้าหากค่าดังกล่าวมีอยู่
• method clear : ลบข้อมูลทั้งหมดออกจาก Set นั่นจะทำให้เซ็ตว่างหลังจากเรียกเมธอดนี้
• method pop : นำข้อมูลออกจาก Set และส่งค่ากลับมา ค่าที่ส่งกลับจากเมธอดนี้ไม่สามารถคาดเดาได้ และถ้าเซ็ตเป็นเซ็ตว่าง เมธอดทำให้เกิดข้อผิดพลาด KeyError ขึ้น

FINDING THE UNION OF SETS

• UNION
 - รวม set ตั้งแต่สอง set ขึ้นไปเข้า
 ด้วยกันเป็น set ใหม่
 - ใช้ method union()
 - Format : set1.union(set2)
 - ใช้ operator |
 - Format : set1 | set2
• Intersection
 - ได้ set ใหม่ที่มีข้อมูลที่เหมือนกับ set
 อื่นๆเท่านั้น
 - ใช้ method intersection()
 - Format : set1.intersection(set2)
 - ใช้ operator &
 - Format : set1 & set2
• Symmetric difference
 - ได้ set ใหม่ที่รวมกันของข้อมูลที่ต่าง
 กันเท่านั้นระหว่างสอง set
 - ใช้ method
 symmetric_difference()
 - Format :
 set1.symmetric_difference(set2)
 - ใช้ operator ^
 - Format : set1 ^ set2
• Subset
 - เพื่อเช็คว่าข้อมูลทุกตัวใน set หนึ่ง
 ปรากฏอยู่ในอีก set หนึ่งจริงหรือไม่
 - ใช้ method issubset()
 - Format : setA.issubset(setB)
 - ใช้ operator <=
 - Format : set1 <= set2
• Superset
 - เพื่อเช็คว่ามีข้อมูลทุกตัวของ set อื่น
 ปรากฏอยู่ใน set นี้หรือไม่
 - ใช้ method issuperset()
 - Format : setA.issuperset(setB)
 - ใช้ operator <=
 - Format : set1 <= set2
